## Are you good at math?

Can we evaulate some question about 5 (C) + (C)(D)? when C = 1 / 5 and D = 15 , and yes of course.

```
// compute 5(C)
// 5(C) = 5 x 1 / 5;

// compute (C) (D)
// (C)(D) = 1 / 5 x 15 = 15 / 5

// and then add it
var c: int = 1 / 5;
var d: int = 15;
var answer: int = ((5 * 1) / 5) + ((15 * 1) / 5);

print(answer);
```
